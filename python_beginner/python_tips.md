# py_tips

## To know functions available for an object

```python
dir(obj)
```

## To get help and info about a function or a class

```python
help(obj)
help(obj.method)
```

## To see the class of an object 

```python
type(str)
```

## To get the memory-address of an obj

```python
id(ojb) # returns it in decimal base
```
